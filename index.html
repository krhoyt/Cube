Image Processing

  • Start - load image to canvas
  • Grayscale - convert image to grayscale
  • Blur - blur image slightly
  • Filters - using CSS filters
  • Canny - edge detection (binary image)
  • Dilate - emphasize edges
  • Threshold - find edges using threshold

Feature Detection

  • Contours - find image contours
  • Polygons - extract polygons from contours
  • Squares - reduce contours to squares
  • Bounds - remove extraneous squares
  • Video - web camera in place of static image
  • Worker - separate thread for image analysis
  • Hough - analysis based on lines
  • Probablistic - use statistics to guess grid
  • Colors - variations in color models

Alternatives

  • Framing - specifying the capture areas
  • Distance - Euclidian color distance
